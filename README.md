# Antivirus_system
Распределенная система проверки файлов на наличие паттернов.
## Компоненты
### Сервер
- Принимает подключения
- Создает отдельный процесс для каждого клиента с помощью fork()
- Проверяет файлы
- Собирает общую статистику между процессами с помощью shared memory
- Обрабатывает запросы статистики через FIFO

### Клиент
- Отправляет файлы на проверку через TCP
- Использует бинарный протокол для передачи данных

### Статистика
Утилита для получения статистики через IPC

- Записывает в FIFO_REQUEST
- Читает из FIFO_RESPONSE
- Выводит статистику в консоль в json формате

### Примечание:
- Файлы загружаются в память (не потоковая обработка!)
- Используются библиотеки POSIX sockets, shared memory
